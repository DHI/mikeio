MIKE IO: input/output of MIKE files in Python¶

Facilitate creating, reading and writing dfs0, dfs2, dfs1 and dfs3, dfsu and mesh files. Reading Res1D data.

Requirements¶

  • Windows or Linux operating system

  • Python x64 3.6 - 3.9

  • (Windows) VC++ redistributables (already installed if you have MIKE)

Installation¶

From PyPI:

pip install mikeio

For Anaconda:

conda install -c conda-forge mikeio

Or development version:

pip install https://github.com/DHI/mikeio/archive/main.zip

Getting started¶

>>>  from mikeio import Dfs0
>>>  dfs = Dfs0('simple.dfs0')
>>>  df = dfs.to_dataframe()
