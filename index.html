MIKE IO: input/output of MIKE files in Python

Read, write and manipulate dfs0, dfs1, dfs2, dfs3, dfsu and mesh files.

See our sister library MIKE IO 1D for .res1d and .xns11 files.

The documentation is currently being updated to reflect the upcoming version of MIKE IO 1.0

Requirements

  • Windows or Linux operating system

  • Python x64 3.7 - 3.10

  • (Windows) VC++ redistributables (already installed if you have MIKE)

Installation

pip install mikeio

Getting started

>>>  import
>>>  ds = mikeio.read('simple.dfs0')
>>>  df = ds.to_dataframe()

Read more in the getting started guide.
