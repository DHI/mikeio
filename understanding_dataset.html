Understanding Dataset¶

The MIKE IO Dataset is a common data structure for data read from dfs files. All read() methods in MIKE IO returns a Dataset with three main properties:

  • items - a list of ItemInfo with name, type and unit of each item

  • time - a pandas.DateTimeIndex with the time instances of the data

  • data - a list of NumPy arrays—one for each item
